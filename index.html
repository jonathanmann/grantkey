Grantkey is a P2P credit and contract platform

Grantkey is an automated clearinghouse platform allowing any party to grant reputation-backed credit which can be exchanged via smart contract by any two parties who trust the issuer.

CONNECT TO NETWORK

Fast

By shifting away from whole-network validation to a trusted guarantor, Grantkey processes transactions at lightning speed with no limits on scalability.

Verifiable

All transactions are digitally signed proving the digital identity of each party. The network guarantor provides an immutable ledger detailing all transactions and balances over a provided time period.

Secure

Each transaction is automatically validated by the mutually trusted guarator upholding digital contracts and preventing double spend attacks.

Web-Based Client

All transactions can be performed in the browser making interaction with the network fast, simple, and secure. Use web client to generate a key pair, store the private key in a secure location, and join the network by connecting to a trusted guarantor.

Credit and Smart-Contracts

Smart-contracts can be used to transfer credit based on the criteria specified in the contract. Since all transactions are automatically validated by a trusted guarantor, contract validation is almost instantaneous.

Verified Immutable Transactions

The guarantor publishes transactions to a blockchain allowing any user to audit the veracity of the network.

Open Source

A trusted open source code base that can be audited for vulnerabilities.
